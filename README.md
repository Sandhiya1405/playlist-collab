# playlist-collab
 web platform for college and hostel events where students can suggest songs and organizers can manage and curate event playlists efficiently.

## Features
- Students can suggest songs
- Organizers can review and manage playlists
- Event-based playlist system
- User login system
- Voting for songs
- Real-time updates

## Tech Stack
HTML, CSS, JavaScript

## How it Works
Users submit song suggestions through the platform. Organizers review these suggestions and curate the final playlist for events.
